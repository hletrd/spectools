### ```spectools``` package

* A simple, flexible Python package for basic spectral analysis.
* Dependencies: ```AstroPy```, ```ccdproc```, ```NumPy```, ```scikit-image```.
 * Developed on Python 3.4+

#### ```spectools.SpecData```

* Class to hold spectral data, and to analyze it.

#### ```spectools.reflamps```

* Data of reference lamps.
